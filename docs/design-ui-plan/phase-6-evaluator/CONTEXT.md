# Phase 6: 评估器展示增强

## 阶段概述

**目标**：优化评估器页面，增强预置评估器展示和自定义评估器引导

**优先级**：P2

**前置依赖**：无

**涉及页面**：`/evaluators` (评估器管理)

## 当前问题

1. 预置评估器卡片可点击性不明确
2. 自定义评估器列表为空时没有引导
3. 评估器类型标签太小
4. 评估器适用场景说明不足

## 优化目标

### 预置评估器展示增强
- 卡片样式优化，增加可点击提示
- 显示适用场景说明
- 查看详情入口明显

### 自定义评估器引导
- 空状态有引导性内容
- 文档链接
- 快速创建入口

### 类型标识增强
- 类型图标
- 颜色区分
